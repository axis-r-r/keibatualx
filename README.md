# バーチャル競馬 RC4.0 — 安定運用版

Supabaseを使わず、Netlify Functionsのサーバー時刻と固定シードで全端末のレースを共通化します。

## RC4.0の重要な修正

- レース進行を絶対時刻の期限で管理し、タブ停止・端末スリープ・描画エラーでも永久停止しない
- 32秒を超えたレースは同じ固定結果で自動決着
- タブ復帰、ウィンドウ復帰、オフライン復帰時に即座に状態を再確認
- 古いRC2.9を表示し続ける原因になり得るService WorkerキャッシュをRC4.0へ更新
- HTMLはネットワーク優先、Netlify Functionはキャッシュ禁止
- 日本時間で毎日第1〜720レース表示

## 更新方法

このフォルダの中身をGitHubの `keibatualx` リポジトリ直下へすべて上書きし、Commit changesを押してください。Netlifyが自動デプロイします。

公開後、最初の一度だけ `Ctrl + Shift + R` で強制更新してください。画面タイトルが「バーチャル競馬 RC4.0」なら更新成功です。
